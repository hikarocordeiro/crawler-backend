# Web Crawler

This is crawler build in Node.

## Installing

 1. Checkout the project
 2. Navigate to project folder
 3. Run the follow command to install
```
yarn
```
4. Run the follow command to build
```
yarn build
```
5. Start the application
```
yarn start
```
